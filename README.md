<p>In this project, an online store has been implemented. In that project, there are various sections including shopping cart,
  orders, product filtering, etc. The important point in this project is that the shopping cart section has been implemented using a session, 
  which is one of the methods of the mod.</p>

  <p>Features</p> <br>
  
  <ul>
    <li>Authentication system (login, rgister, logout)</li>
    <li>Shoping history</li>
    <li>Coupon system </li>
    <li>Back ground process(celery and rabbitmq)</li>
    <li>Recommender system(Redis)</li>
  </ul>
